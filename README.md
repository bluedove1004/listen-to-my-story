# 🌸 내 이야기를 들어줘!

24시간 언제든지 이야기를 들어주는 AI 심리 상담 웹앱입니다.

## 프로젝트 소개

"내 이야기를 들어줘!"는 Google Gemini API를 활용한 심리 상담 챗봇입니다.  
따뜻하고 공감적인 상담사 **소담이**가 언제든지 여러분의 이야기를 들어드립니다.

## 사용 기술

| 기술 | 설명 |
|------|------|
| HTML5 | 앱 구조 |
| CSS3 | 스타일 (파스텔 컬러, 반응형 디자인) |
| JavaScript | 앱 동작 로직 |
| Gemini API | AI 심리 상담 응답 (gemini-2.0-flash 모델) |
| Google Fonts | Noto Sans KR 폰트 |

## API 키 발급 방법

1. [Google AI Studio](https://aistudio.google.com/) 접속
2. Google 계정으로 로그인
3. 상단 메뉴에서 **"Get API key"** 클릭
4. **"Create API key"** 버튼 클릭
5. 생성된 API 키 복사

## 실행 방법

별도 설치나 서버 없이 브라우저에서 바로 실행할 수 있습니다.

1. `index.html` 파일을 브라우저(Chrome, Safari, Edge 등)로 열기
2. Gemini API 키 입력
3. **상담 시작하기** 버튼 클릭
4. 소담이와 대화 시작!

## 파일 구조

```
내 이야기를 들어줘!/
├── index.html       # 메인 페이지 (전체 화면 구조)
├── css/
│   └── style.css    # 전체 스타일 (색상, 레이아웃, 애니메이션)
└── js/
    ├── api.js       # Gemini API 통신 모듈 (GeminiAPI 클래스)
    └── app.js       # 앱 동작 로직 (화면 전환, 메시지 처리)
```

## 주요 기능

- **24시간 상담**: 언제든지 소담이와 대화할 수 있습니다
- **공감 중심 응답**: 판단 없이 경청하고 공감하는 대화
- **대화 기록 유지**: 같은 세션에서 이전 대화 내용을 기억합니다
- **API 키 자동 저장**: 로컬 스토리지에 저장되어 다음에 자동 로그인
- **대화 초기화**: 설정 메뉴에서 새로 시작할 수 있습니다
- **위기 배너**: 항상 상단에 위기상담 전화번호를 표시합니다
- **반응형 디자인**: 모바일과 PC 모두 지원합니다

## 주의사항

### 앱의 한계
- 소담이는 AI 도우미로, **전문 심리치료사나 의사를 대체하지 않습니다**
- 심각한 정신건강 문제는 반드시 전문가의 도움을 받으세요
- AI의 응답이 항상 정확하거나 적절하지 않을 수 있습니다

### 위기 상황 전문 기관 연락처

| 자살예방상담전화 | **1393** | 24시간 |
| 정신건강위기상담전화 | **1577-0199** | 24시간 |
| 생명의전화 | **1588-9191** | 24시간 |

### API 키 보안
- API 키는 **브라우저 로컬 스토리지에만 저장**됩니다
- 외부 서버로 전송되지 않으며, 브라우저를 통해 Gemini API에 직접 연결됩니다
- API 키가 노출되지 않도록 주의하세요
